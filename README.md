# ハローワールド

シンプルで美しいデザインの「ハローワールド」ページです。

## 特徴

- モダンなグラデーション背景
- レスポンシブデザイン
- スムーズなアニメーション効果

## GitHub Pagesでの公開方法

1. このリポジトリをGitHubにプッシュ
2. リポジトリの「Settings」→「Pages」に移動
3. 「Source」で「Deploy from a branch」を選択
4. 「Branch」で「main」を選択し、フォルダは「/ (root)」を選択
5. 「Save」をクリック

数分後、以下のURLでアクセスできます：
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/ハローワールド.html`

## ローカルでの実行方法

```bash
python3 -m http.server 8001
```

ブラウザで `http://localhost:8001/ハローワールド.html` にアクセスしてください。
